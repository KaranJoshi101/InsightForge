# Admin Dashboard Guide

Welcome to the Admin Dashboard! This guide will help you quickly understand and efficiently use the main features available to administrators.

---

## 1. Login
- Access the dashboard via the admin login page.
- Enter your admin credentials to sign in.

## 2. Dashboard Overview
- The dashboard provides a summary of key metrics and quick links to manage surveys, users, and training content.

## 3. Managing Surveys
- **View Surveys:** See all existing surveys, their status, and basic stats.
- **Create Survey:** Click the "Add Survey" button to create a new survey. Fill in the required fields and save.
- **Edit Survey:** Click the edit icon next to a survey to update its details.
- **Delete Survey:** Click the delete icon to remove a survey. Confirm the action when prompted.
- **Publish/Unpublish:** Use the publish toggle to make a survey visible or hidden to users.

## 4. Managing Users
- **View Users:** Access the user list to see all registered users.
- **Ban/Unban Users:** Use the ban toggle to restrict or restore user access.
- **Edit User Info:** Click the edit icon to update user details.

## 5. Training Content
- **View Training:** See all training modules and videos.
- **Add Training:** Click "Add Training" to upload new content.
- **Edit/Delete:** Use the respective icons to update or remove training items.

## 6. Media & Responses
- **Media:** Manage uploaded media files and link them to surveys or training.
- **Responses:** View survey responses and export data if needed.

## 7. Settings & Security
- **Profile:** Update your admin profile and password from the settings menu.
- **Logout:** Always log out after your session for security.

---

## Tips for Efficiency
- Use the search bars and filters to quickly find surveys, users, or training items.
- Confirm actions (like delete or ban) to avoid accidental changes.
- Check dashboard notifications for important updates or alerts.
